#### [206.反转链表]([https://leetcode.cn/problems/reverse-linked-list/description])

##### 思路：将当前cur的指针本是指向cur的后一个，咱们让他指向cur的前一个,即pre。 pre后移，cur后移
```c

class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* pre=nullptr;
        ListNode* cur=head;
        while(cur!=nullptr){
            ListNode* q=cur->next;//找到下一个位置，防止断链
            cur->next=pre;//开始翻转
            pre=cur;//将前驱指针后移
            cur=q;//cur也后移
        }
        return pre;
    }
};


```

#### [160.相交链表]([https://leetcode.cn/problems/intersection-of-two-linked-lists/description/])

##### 思路：找到相交点，两个链表同时开始移动，每次都移动一步，让A走完A的路后，走B路，让B走完B的路后，走A路。最后肯定也是同时到达终点。也就是说到达相交点后的每一个位置都是一起走完的。
所以只需按照上述要求来模拟过程，当也就是两个指针相等时，就找到了相交点。
```c

class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        ListNode*p=headA;
        ListNode*q=headB;
        while(p!=q){
             if(p==NULL){//如果A路走完了，走B路
                p=headB;
            }else{
                p=p->next;
            }
            if(q==NULL){//如果B路走完了，走A路
                q=headA;
            }else{
                q=q->next;
            }
        }
        return p;
    }
};

```
