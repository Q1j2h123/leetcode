# 滑动窗口

## 1.固定大小的滑动窗口

### 1.1 思想
从一开始就设定一个固定大小的窗口进行滑动
#### [438. 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string)
##### 思想/解法：每一次窗口滑动时，都需比较p中的字符是否相同，则单独开一个p数组记录p的字符 保持不变。再维护一个固定大小为PLen的窗口向右滑动，每一轮滑动一个字符。

```c
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        int sLen=s.size();
        int pLen=p.size();
        vector<int>ans;
        if(pLen>sLen)return vector<int>();
        vector<int>Pcnt(26,0);
        vector<int>Scnt(26,0);
        for(int i=0;i<pLen;i++){
            Pcnt[p[i]-'a']++;//记录p中字符 并保持不变
            Scnt[s[i]-'a']++;//同时记录s中前Plen个字符（也就是固定了滑动窗口的大小，之后每一轮只向右滑动一个单位）
        }
        if(Scnt==Pcnt)ans.push_back(0);//如果刚开始就相同，则将0号位置记录到答案中
        for(int i=0;i<sLen-pLen;i++){
            Scnt[s[i]-'a']--;//左窗口移出
            Scnt[s[i+pLen]-'a']++;//右窗口移入字符 （注意是始终保持Plen的窗口大小s[i+pLen]）
            if(Scnt==Pcnt)ans.push_back(i+1);
        }
        return ans;
    }
};

```
